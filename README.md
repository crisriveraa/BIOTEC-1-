# CRISPR-Cas9: The Molecular Scissors

An interactive scientific article that explains the fundamentals, mechanism,
applications, limitations, and ethical considerations of CRISPR-Cas9 gene
editing.

## Live website

Explore the published article:

**[crisriveraa.github.io/BIOTEC-1-](https://crisriveraa.github.io/BIOTEC-1-/)**

## Project overview

CRISPR-Cas9 is one of the most influential technologies in modern molecular
biology. This project presents the science behind the system through accessible
writing, interactive visuals, and a responsive web experience.

The article was created as the first project in my biotechnology portfolio. Its
purpose is to communicate a complex scientific topic clearly while
demonstrating research, scientific communication, web development, and visual
design skills.

## What the article covers

- An introduction to CRISPR-Cas9
- The roles of the Cas9 enzyme and guide RNA
- A five-step explanation of the editing mechanism
- Applications in therapeutic medicine
- Applications in agriculture
- Applications in biological research
- Applications in molecular diagnostics
- Scientific limitations, including off-target editing and delivery
- Ethical questions concerning heritable editing, access, and governance
- Scientific references and further reading

## Interactive features

- Animated DNA double helix in the opening section
- Scroll-based presentation of the CRISPR-Cas9 mechanism
- Navigation buttons for each application
- Original editorial illustrations created with HTML and CSS
- Responsive layout for desktop and mobile devices
- Reduced-motion support for accessibility
- Direct links between in-text citations and the reference list

## Technologies used

- **HTML5** for the article's structure and scientific content
- **CSS3** for visual design, responsive layouts, and animations
- **JavaScript** for the animated genome and scroll interactions
